### href 和 src 的差別

-   請求資源類型不同  
    （1）href 指向網路資源所在位置，建立和當前元素（錨點）或當前文檔（連結）之間的聯繫。  
    （2）在請求 src 資源時會將其指向的資源下載並應用到文檔中，比如 JavaScript 腳本，img 圖片。
