#### 物件縮寫(Object shorthand)

-   屬性縮寫 : 當物件內的 Key 跟 Value 同名時，可以不用重複撰寫  
    `function ( x, y ){ return { x:x, y:y } }` =>
    `function ( x, y ){ return { x, y } }`

*   計算屬性 : 當宣告 object 時，想要動態指定物件內的 Key，更方便的寫法  
    `function() {`  
    `onst obj = {};`  
    `obj[key] = value;`  
    `return obj;`  
    `}`
